## 🛠 想改代码 / 一起做开发？（Windows 新手照着做，全程点鼠标，不用敲命令）

游戏的全部代码就在 **一个文件 `index.html`** 里。要改它、并在自己电脑上看效果，最省事的办法是用 **VS Code + Live Server 插件**：

### 第 1 步：装 VS Code（免费的代码编辑器）
打开 👉 https://code.visualstudio.com/ ，点蓝色大按钮下载 **Windows 版**，下载后双击安装，一路点「下一步」。

### 第 2 步：把项目弄到自己电脑
- **只想看看 / 改着玩**：打开 👉 https://github.com/Starsky618/drop ，点绿色 **`< > Code`** 按钮 → **`Download ZIP`**，下载后**解压**到桌面。
- **要正式一起协作**：在那个页面右上角点 **`Fork`**（复制一份到你自己的 GitHub 账号），再 Download ZIP。

### 第 3 步：在 VS Code 里打开并运行
1. 打开 VS Code → 左上角 **「文件」→「打开文件夹」** → 选刚解压出来的 `drop` 文件夹
2. 点左边竖排的**扩展图标**（四个小方块那个），搜索框输入 **`Live Server`**，点 **Install** 装上
3. 装好后，在左边文件列表里**右键点 `index.html`** → 选 **`Open with Live Server`**
4. 浏览器会自动打开游戏。**之后你改代码、按 Ctrl+S 保存，浏览器会自动刷新**——边改边看效果

> ⚠️ **重要**：千万**别直接双击 `index.html`**（会白屏！）。一定要用上面的 Live Server 方式打开——这是网页游戏的技术限制，不是 bug。

## 🧩 技术栈（好奇的看）
纯网页，一个 `index.html` 搞定，没有复杂安装。用 Three.js 画 3D、WebAudio 合成音效，全是代码生成的，没有大图片大文件，整个游戏体积很小。
