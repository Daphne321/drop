# 房间家具清单（每种房间放什么 / 怎么拼 / 埋什么科普）

从零程序化建房间时查这份。每个房间给：**家具清单 + 拼法 + 地震破败细节 + 色调材质 + 科普埋点**。

DROP 是地震逃生科普游戏，所以每个房间不只是装饰——**埋一条"地震时这里危险还是安全"的知识**，让探索本身在教救命常识。这是房间的灵魂，别只堆家具。

## 通用家具建模技法

- **多 box 拼，绝不一个大方块糊弄**：一件家具拆成结构件（如桌=台面+4腿+横撑）。这是这次客厅"精致感"的来源。
- **圆柱用 `CylinderGeometry`**：杯子/灶头/花盆/管道/瓶罐/人腿。
- **歪斜 = 地震感**：给家具/物件 `.rotation.z` 或 `.rotation.set(...)` 加一点角度（电视歪、椅子翻、画框斜），瞬间有"被震过"的真实感。
- **散落物撒法**：循环撒一批小 box（`s=0.04~0.17`），随机 position + 随机 rotation，灰白色，撒地面——"地震后狼藉"的地面层。
- **家具全用 `deco()`**（不碰撞），靠墙摆，留出门洞和走道。
- **色调统一**：服从全局"神秘暗灰"，靠 1~2 盏暖 `PointLight` 给焦点。别自己加全局光。

## 厨房 Kitchen
- **家具**：下橱柜(长 box)+台面(扁 box，浅木/石英)+上吊柜+灶台(box+4 个 CylinderGeometry 灶头)+抽油烟机(斜 box)+冰箱(高 box+竖把手 metalMat)+水槽(box+内凹深色+金属水龙头)+餐边柜。
- **破败**：地上打翻的锅(倒扣 cylinder)、碎盘子(小白 box 散落)、吊柜门歪开(rotation.y)、燃气软管垂下。
- **色调材质**：瓷砖墙 `tileTexture('#e8e6e0','#9a968c')` + 不锈钢 `metalMat` + 木橱柜 `woodTexture('#a39788')`。
- **光**：1 盏暖顶灯。
- **🔴 科普埋点**：厨房是地震高危区——燃气泄漏+明火=爆炸、刀具和重锅坠落、玻璃。可在灶台埋"先关燃气总阀"的细节或提示。

## 卫生间 Bathroom
- **家具**：马桶(底座 box+水箱 box+椭圆盖)+洗手台(台 box+圆盆 cylinder+水龙头 metalMat)+镜子(薄 glassMat box+chrome 框)+淋浴隔断(glassMat 大薄板+花洒 cylinder)+毛巾架+置物架(瓶瓶罐罐小 cylinder)。
- **破败**：碎裂镜子(裂纹纹理/几道反光细 box)、墙面漏水痕、瓷砖掉落露暗斑、倒下的洗漱用品。
- **色调材质**：冷白瓷砖 `tileTexture('#dfe6e6','#8a9090')`，整体冷调。
- **光**：镜前灯可用冷白 `PointLight(0xeef2f6,...)`（卫生间冷光合理）。
- **🟢 科普埋点**：小开间+承重墙多，地震时是家里相对安全的躲避点之一——但要远离镜子/玻璃淋浴房/热水器。这是正面知识(和阳台形成对比)。

## 阳台 Balcony
- **家具**：栏杆(立柱+横杆，metalMat 或木)+晾衣架+晾着的衣物(布料薄平面，给点垂感/晃动)+花盆植物(cylinder 盆+IcosahedronGeometry 叶)+旧藤椅+空调外机(box+格栅细 box)+堆的杂物。
- **破败**：倒下摔碎的花盆(土撒一地)、掉落/挂歪的衣物、裂开的地砖、栏杆松动。
- **色调材质**：半室外——比室内稍亮(暖灯强一点+一点冷天光感)，水泥地 `makeConcrete` + 栏杆金属。朝外可露一点点天光(但别建整个室外，那是结局场景的活)。
- **🔴 科普埋点**：地震时绝对别往阳台跑！——悬挑结构最易塌、玻璃和高空坠物。玩家走进去应该"隐隐觉得不该来"。反面教材。

## 书房 Study
- **家具**：高书架(一排排彩色书脊小 box，塞满)+书桌(台面+腿)+办公椅+台灯(底座+杆+锥罩+暖 PointLight)+成摞的书+地球仪(球+支架)+电脑(屏 screenMat+键盘 box)。
- **破败**：书散落一地、椅子翻倒、**书架整个倾倒**(rotation 大角度，压在地上/桌上)。
- **色调材质**：暖木 `woodTexture('#857769')`+暖灯，温馨被毁的反差。
- **🔴 科普埋点**：高大书架/柜子地震时会倒下压人——要固定在墙上，或躲避时远离。直接呼应游戏的"被压死"主题。

## 儿童房 KidsRoom
- **家具**：小床(矮 box 床架+床垫+小被子布料)+玩具箱+散落积木(一堆小彩色 box)+书包+矮书架+毛绒玩具(球+box 拼的熊)+墙上贴画(彩色薄 box)。
- **破败**：玩具散落满地、小床移位、矮柜倒。
- **色调材质**：明快色(粉/黄/蓝)——被暗灰氛围一压，反差揪心。
- **💔 科普埋点**：情感冲击点(孩子的房间被毁)+可埋"地震时护住孩子头颈、躲到结实家具旁"。

## 储藏室 / 杂物间 Storage
- **家具**：纸箱堆(box 堆叠，歪斜)+金属货架(metalMat 立柱+层板，塞杂物)+旧家电(box)+工具+旧行李箱+一只裸灯泡(小球+暖 PointLight)。
- **破败**：箱子倒塌散落、货架歪倒、杂物挡路。
- **色调材质**：昏暗+单个暖灯泡，最压抑的房间。
- **🔴 科普埋点**：堆高的杂物/货架地震时倒塌会砸人+堵住逃生路——别在通道堆东西。

## 楼道 / 楼梯间 Corridor
- **家具**：信箱墙(一排排小金属门 box)+电表箱+堆放的自行车(细 cylinder 轮+frame box)+灭火器(红 cylinder)+堆的杂物+应急灯。
- **破败**：掉落的天花板碎块、晃动/坏掉的灯、堆物半堵通道。
- **色调材质**：水泥 `makeConcrete`+冷调应急灯。
- **🔴 科普埋点**：逃生通道堆杂物=地震时堵死逃生路。灭火器/应急灯是"对的细节"。
